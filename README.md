# TravelVista
A responsive travel website with login, popular destinations, about and contact pages built with HTML, CSS, and little JavaScript.
Simple Gmail Login Validation
Description

This project is a simple login form using HTML, CSS, and JavaScript. It validates user input and provides warnings for incorrect credentials. The login rules are:

Email must be a @gmail.com address.

Password must be more than 6 characters.

Shows appropriate warning messages if the input is invalid.

Features

Real-time validation of email and password.

Displays success message for valid Gmail accounts with valid passwords.

Displays error messages for wrong credentials or short passwords.

How to Use

Open index.html in a browser.

Enter a Gmail email and password.

Click the Login button.

Check the message below the input fields for success or error messages.
